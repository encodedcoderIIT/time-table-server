Key Functionalities of the Time Table Server

  • Push Notifications: The server sends push notifications to subscribed users about upcoming classes.
  • Subscription Management: The server handles subscription and unsubscription requests from users.
  • Timetable Management: The server manages and serves the timetable data to the client.
  • Service Worker Registration: The server helps in registering the service worker for push notifications.
  • VAPID Key Management: The server manages VAPID keys for secure push notifications.
